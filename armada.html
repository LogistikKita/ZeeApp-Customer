Armada Kami: Detail Kendaraan & Kapasitas

Pilih kendaraan yang tepat. Jaminan pengiriman aman dan efisien untuk setiap jenis kargo Anda.

Pickup Bak Terbuka

Cepat dan ideal untuk kargo domestik ringan atau distribusi dalam kota.

  • Kapasitas Maks:± 1.300 Kg
  • Volume Maks:± 2 CBM
  • Dimensi (P x L x T):2.4m x 1.4m x 1.4m

Pickup Box

Proteksi maksimal untuk barang berharga dan sensitif cuaca.

  • Kapasitas Maks:± 1.000 Kg
  • Volume Maks:± 3 CBM
  • Dimensi (P x L x T):2.4m x 1.4m x 1.4m

Blind Van

Kendaraan ramping untuk pengiriman cepat di area padat.

  • Kapasitas Maks:± 600 Kg
  • Volume Maks:± 3 CBM
  • Dimensi (P x L x T):± 2.0m x 1.3m x 1.3m

CDD (Box/Bak)

Andalan untuk distribusi massal jarak menengah (FTL/LTL).

  • Kapasitas Maks:4.000 - 5.000 Kg
  • Volume Maks:15 - 20 CBM
  • Dimensi (P x L x T):± 4.5m x 2.0m x 2.0m

Fuso (Box/Bak)

Kapasitas angkut besar untuk proyek dan kargo FTL jarak jauh.

  • Kapasitas Maks:7.000 - 8.000 Kg
  • Volume Maks:25 - 30 CBM
  • Dimensi (P x L x T):± 7.5m x 2.4m x 2.4m

Tronton Bak

Angkutan super berat dan kargo dimensi besar yang tidak memerlukan penutup.

  • Kapasitas Maks:10.000 - 15.000 Kg
  • Volume Maks:35 - 50 CBM
  • Dimensi (P x L x T):± 9.5m x 2.4m x 2.4m

Tronton Wingbox

Akses bongkar muat 3 sisi (atas/samping) untuk kecepatan logistik.

  • Kapasitas Maks:10.000 - 15.000 Kg
  • Volume Maks:40 - 55 CBM
  • Dimensi (P x L x T):± 9.5m x 2.5m x 2.5m

Trailer Flatbed / Container

Solusi angkutan terberat dan kontainer (20ft & 40ft) antar pelabuhan.

  • Kapasitas Maks:20.000 - 30.000 Kg
  • Tipe Kontainer:20ft / 40ft
  • Panjang Maks:± 12.0m
